# Sales-Analytics-Dashboard-Power-BI-Project

## 📌 Project Overview

This project presents an interactive **Sales Analytics Dashboard** built using **Microsoft Power BI**. The dashboard transforms raw sales data into meaningful business insights, enabling decision-makers to monitor performance, identify trends, and make data-driven decisions.

---

## 🎯 Business Problem

Businesses generate large volumes of sales data every day, making it difficult to quickly identify:

- Which products generate the highest sales?
- Which product lines are most profitable?
- How are sales performing year over year?
- Which regions contribute the most revenue?
- What is the overall business performance?

This dashboard solves these challenges by providing an interactive and easy-to-understand visualization of sales performance.

## 📊 Business Insights

- Classic Cars generated the highest sales.
- 2004 recorded the highest quantity ordered.
- Total sales exceeded **$10 Million**.
- Profit reached **$3.83 Million**.
- The company maintained a **39.84% Profit Margin**.
- Global sales distribution can be analyzed using the interactive map.
- Users can filter data by:
  - City
  - Employee
  - Product Line
  - Order Year
